# Skill: Sensual Design — Softness, Intimacy, Elegance

## Purpose

This skill teaches how to create interfaces that feel tactile, warm, and intimate. Not "tech UI" — something you want to touch, linger on, and explore slowly. The visual equivalent of velvet, candlelight, and whispered conversation.

## When to Use

- When any element feels "cold" or "corporate"
- When reviewing overall emotional tone
- When the user says "make it more premium" or "more elegant"
- When something feels like a SaaS dashboard instead of a gallery

## Core Qualities

### 1. Softness

Nothing is hard. Edges dissolve. Boundaries are suggestions.

**Implementation:**
- No hard borders — ever. If separation is needed, use space or tone shifts
- No sharp box shadows — only diffuse ambient light if needed
- Images fade at edges rather than having hard crops (CSS mask or gradient overlay)
- Text color is never pure white on pure black — always warm tones with imperfect contrast

```css
/* Soft edge mask for images — dissolves the boundary */
.image-soft-edge {
  mask-image: linear-gradient(
    to bottom,
    transparent 0%,
    black 8%,
    black 92%,
    transparent 100%
  );
  -webkit-mask-image: linear-gradient(
    to bottom,
    transparent 0%,
    black 8%,
    black 92%,
    transparent 100%
  );
}

/* Soft horizontal fade for images that bleed off-screen */
.image-bleed-right {
  mask-image: linear-gradient(
    to right,
    black 0%,
    black 75%,
    transparent 100%
  );
}
```

### 2. Slow Interaction

Everything responds, but never instantly. There is always a moment of anticipation.

**Timing rules:**
- Hover state transitions: 500ms minimum
- Image color reveal: 800ms
- Opacity changes: 600ms
- Nothing "snaps" into place — everything eases

```css
/* Slow, sensual hover response */
.sensual-element {
  transition: all 700ms cubic-bezier(0.16, 1, 0.3, 1);
}

/* Image that slowly wakes up on hover */
.sensual-image {
  filter: grayscale(0.7) brightness(0.9);
  transition: filter 900ms cubic-bezier(0.16, 1, 0.3, 1);
}
.sensual-image:hover {
  filter: grayscale(0) brightness(1);
}
```

### 3. Warmth

The color system must evoke warmth, not technology.

**Temperature palette:**
```css
/* Warm blacks — never cold */
--bg-silk:    #0d0b09;  /* silk in candlelight */
--bg-shadow:  #0a0908;  /* warm shadow */
--bg-ember:   #0d0907;  /* faint ember glow */

/* Warm whites — never clinical */
--text-cream:    #F0EBE3;  /* aged paper */
--text-whisper:  #C8C0B4;  /* distant, secondary */
--text-ghost:    #8A837A;  /* barely there */

/* Accent — champagne, not gold */
--accent:     #C4A88A;  /* champagne warmth */
--accent-dim: #9E8A72;  /* muted champagne */
```

### 4. Elegance Through Restraint

Elegance is not about adding beauty — it is about removing everything that is not beautiful.

**Rules:**
- One typeface family only (serif for display, sans for body — same family feel)
- One accent color only
- One animation style only (same easing everywhere)
- If you add an element and the composition doesn't improve, remove it

### 5. Tactile Quality

The interface should feel like a physical object — something with weight, texture, and surface.

**Techniques:**
- Subtle noise texture overlay on backgrounds (2-3% opacity) for surface feel
- Micro-gradients on large dark surfaces (not flat color, a very gentle light falloff)
- Image containers that feel like they are recessed into the surface

```css
/* Subtle surface texture — adds physical quality */
.surface-texture::after {
  content: '';
  position: absolute;
  inset: 0;
  background-image: url('/assets/noise.png');
  background-repeat: repeat;
  opacity: 0.025;
  pointer-events: none;
  mix-blend-mode: overlay;
}

/* Micro-gradient on large dark areas — prevents flatness */
.surface-gradient {
  background: radial-gradient(
    ellipse at 30% 40%,
    #0f0d0a 0%,
    #0a0908 70%
  );
}
```

## Forbidden Elements

These break the sensual quality:

- ❌ Sharp 1px borders
- ❌ Box shadows
- ❌ Saturated colors (anything with saturation > 15%)
- ❌ Pure white text (#fff)
- ❌ Pure black backgrounds (#000)
- ❌ Angular geometric shapes
- ❌ Icon libraries (use text or custom marks)
- ❌ Buttons that look like buttons (use text links)
- ❌ Loading indicators that feel mechanical
- ❌ System fonts
- ❌ Tooltips
- ❌ Anything that reminds the user they are using a computer

## The Sensual Test

Before shipping:

1. Would this feel right printed on heavy cotton paper?
2. Could this exist in a dimly lit gallery?
3. Does this invite slow exploration or quick scanning?
4. If I touched the screen, would I expect to feel something?
5. Does this feel like it was made by a person, or generated by a system?
